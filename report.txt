
ОТЧЕТ ПО ЛАБОРАТОРНОЙ РАБОТЕ 2: RNN ДЛЯ БИОМЕТРИЧЕСКОЙ ИДЕНТИФИКАЦИИ
================================================================================

1. ОПИСАНИЕ ДАТАСЕТА
   - Датасет: LFW (Labeled Faces in the Wild)
   - Количество изображений: 1288
   - Количество классов (персон): 7
   - Размер изображения: 50x37 пикселей
   - Тип биометрии: Статическая (изображения лиц)

2. АРХИТЕКТУРА RNN
   - Тип: SimpleRNN (простая рекуррентная сеть)
   - Количество RNN слоев: 2
   - Размеры RNN: 64 -> 32 нейронов
   - Dropout: 0.2-0.3 для регуляризации
   - Активация: tanh (RNN), relu (Dense), softmax (выход)
   - Входные данные: Изображение обрабатывается как последовательность строк пикселей

3. ОБУЧЕНИЕ
   - Процент обучающей выборки: 70%
   - Процент тестовой выборки: 30%
   - Оптимизатор: Adam
   - Функция потерь: Categorical Crossentropy
   - Early Stopping: Да (patience=10)
   - Learning Rate Reduction: Да (factor=0.5, patience=5)

4. РЕЗУЛЬТАТЫ ОБУЧЕНИЯ
   - Финальная точность на обучающей выборке: 0.5105
   - Финальная точность на валидационной выборке: 0.5426
   - Признаки переобучения: -0.0321

5. МЕТРИКИ КАЧЕСТВА
   - Accuracy: 0.5426
   - Recall (Macro): 0.3058
   - F1-Score (Macro): 0.2759
   - TPR (True Positive Rate): 0.5426
   - FPR (False Positive Rate): 0.0762
   - MSE (Mean Squared Error): 0.0818
   - MAE (Mean Absolute Error): 0.1725
   - Mean AUC: 0.8046

6. МАТРИЦА ОШИБОК (усредненные значения)
   - TP (True Positives): 30.00
   - TN (True Negatives): 306.43
   - FP (False Positives): 25.29
   - FN (False Negatives): 25.29

7. ВЫВОДЫ

   7.1. Эффективность RNN для биометрии:
        - RNN показывает умеренную эффективность для распознавания лиц
        - Обработка изображения как последовательности строк частично работает
        - Модель обучилась выделять некоторые паттерны в данных
        - Для изображений CNN будет эффективнее

   7.2. Качество модели:
        - Accuracy 0.5426 указывает на умеренное качество классификации
        - F1-Score 0.2759 показывает несбалансированную работу (дисбаланс классов)
        - AUC 0.8046 показывает хорошую разделимость классов (>0.8 = хорошо)

   7.3. Переобучение:
        - Разница между train и validation accuracy составляет -0.0321
        - Переобучение минимально благодаря Dropout и Early Stopping

   7.4. TPR vs FPR:
        - TPR = 0.5426 - модель корректно распознает значительную часть положительных примеров
        - FPR = 0.0762 - низкий уровень ложных срабатываний
        - Соотношение TPR/FPR = 7.12 - отличный баланс

   7.5. Практическое применение:
        - Модель может использоваться для систем контроля доступа
        - Подходит для верификации личности в биометрических системах
        - Требуется дополнительная настройка для промышленного использования

   7.6. Рекомендации по улучшению:
        - Увеличение датасета для лучшей генерализации
        - Использование аугментации данных
        - Экспериментирование с гибридными CNN+RNN архитектурами
        - Применение attention механизмов
        - Тонкая настройка гиперпараметров

8. ЗАКЛЮЧЕНИЕ
   SimpleRNN архитектура продемонстрировала умеренную эффективность
   в задаче биометрической идентификации по изображениям лиц. Модель успешно
   обучилась и показала хорошие результаты (AUC 0.81), несмотря на дисбаланс классов.
   Однако RNN не является оптимальным выбором для пространственных данных (изображений),
   так как обрабатывает их как последовательности строк, а не учитывает
   двумерную структуру. Для данной задачи рекомендуется использовать CNN
   (сверточные нейронные сети), которые специально разработаны для обработки
   изображений и учитывают их пространственную структуру.
